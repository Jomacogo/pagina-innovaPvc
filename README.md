## 🚀 Descripción

Aplicación web desarrollada con **React** para la visualización de productos, diseñada para ofrecer una experiencia moderna, rápida y accesible a los clientes.

Permite explorar un catálogo digital de productos y cuenta con funcionalidades de administración para gestionar contenido de manera eficiente.

---

## 💡 Valor del proyecto

* Mejora la experiencia de visualización de productos
* Reduce la necesidad de catálogos físicos
* Facilita la gestión de productos para administradores
* Base escalable para e-commerce futuro

---

## ▶️ Instalación y ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/tu-usuario/pagina-innovaPvc.git
```

2. Entrar al proyecto:

```bash
cd pagina-innovaPvc
```

3. Instalar dependencias:

```bash
npm install
```

4. Ejecutar el proyecto:

```bash
npm run dev
```
